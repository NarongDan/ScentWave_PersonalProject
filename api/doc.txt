    -firstName
    -lastName
    -password // a-zA-Z0-9/ at least 6 characters
    -confirmPassword
    -email
    -phone // 10 characters
    -address

 200 message > data

----Add products onto web application

POST /products
HEADER
    Authorization Bearer accessToken
    
BODY 
    -productName
    -productCost
    -productPrice
    -productImage
    -productDetail

 200 message > data
 400 message > Please insert all fields of product
 500 message

----  Update products on web application

PATCH /products/:productId
HEADER
    Authorization Bearer accessToken
BODY    
    -productName
    -productCost
    -productPrice
    -productImage
    -productDetail

200 message > product updated
500 message
